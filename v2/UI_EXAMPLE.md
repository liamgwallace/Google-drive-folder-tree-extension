## Interactive Elements

1. **Root Folder Items**
   - Click: Switch to this root
   - Right-click: Show delete menu
   - Hover: Gray background

2. **+ Button**
   - Click: Open add root modal
   - Hover: Gray background, blue icon

3. **Active Indicator**
   - Blue background
   - Blue folder icon
   - Bold text (optional)

4. **Delete Confirmation**
   - Browser native confirm() dialog
   - Shows warning if deleting current root
   - Prevents deleting last root
