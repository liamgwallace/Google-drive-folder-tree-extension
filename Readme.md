# Google Drive Tree Navigator

A Chrome extension that adds Google Drive folder navigation to any webpage.

## Features

- Browse your Drive folder hierarchy
- Search files quickly
- Recent files list
- Pin folders as root locations
- Dark mode support
- Per-tab ON/OFF toggle
- Resizable sidebar

## Quick Start

1. Set up OAuth credentials in [Google Cloud Console](https://console.cloud.google.com)
2. Enable the Google Drive API
3. Add your Client ID to `v2/manifest.json`
4. Load the extension: `chrome://extensions/` → Load unpacked → Select `v2` folder

See [Setup Guide](./v2/SETUP_V2.md) for detailed instructions.

## Usage

1. Click the extension icon or use the toggle in the icon bar
2. Sign in with Google
3. Browse folders, search files, or access recent items
4. Pin the sidebar to keep it open while working

## Requirements

- Chrome 114+
- Google Account
- OAuth 2.0 credentials from Google Cloud Console
